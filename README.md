# youtube-manager-python
Python CLI project to manage YouTube videos.

This is a simple command-line YouTube manager built using Python.
It allows users to add, update, delete and view video details using file-based storage.

## Features
- Add videos
- Update video details
- Delete videos
- View all saved videos

## Tools Used
- Python
- JSON (file storage)

## What I Learned
- File handling in Python
- Working with lists and dictionaries
- Writing menu-driven CLI programs
